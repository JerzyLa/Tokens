1. Install

   cd MultiSigWallet
   npm install

2. Compilation

   truffle compile

3. Test

   3.1 Install ganache-cli

   npm install -g ganache-cli

   3.2 In seperate terminal run ganache-cli

   ganache-cli

   3.3 Run JS tests

   truffle test

4. Migrate to network

   truffle migrate --network <networktype>
